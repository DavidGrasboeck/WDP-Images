Images can also be set as a background for basically everything. With background-image the background of this section can be easily set to any image.

background-image: url("./backgroundImages.jpg");

The zoom of the background can be changed with background-size. In this example, the background is 150% of the original size. The size can also be set to fixed pixels size, to cover, to cover the entire object or to contain, to be completely contained within the object.

background-size: 150%;

It can also be a good idea to have a repeating background. The background-repeat mode is on repeat at default. If you don't want that, you can use no-repeat.

background-repeat: repeat;

To create this cool effect, the background-attachment is set to fixed. This means that the background stays at the same spot when you scroll and doesn't move with the page.

background-attachment: fixed;
