In this example, a flexbox is used to always center images in the container.
With display: flex in the container, the item position becomes flexible and can be set with align-items and justify-content.
The flex-direction lets you choose how the items are aligned (column, row).

Try changing your screen width to see the behavior of the flexbox.

.flex-container {
display: flex;
align-items: center;
justify-content: center;
flex-direction: column;
}
.flex-item {
flex: auto;
object-fit: contain;
}
