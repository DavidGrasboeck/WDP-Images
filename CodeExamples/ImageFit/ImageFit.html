Lets say we have following image that we want to include in our website:

There are different ways to do that. This Code example will show the different ways to fit it into a dynamically sized object using object-fit . Here for example: a box that scales its width with screen-size, but has a fixed height. Try changing your screen width to fully compare the options. It is also important to set the images width and height to 100% to make it dependent on the parent.

To fully fill the box the default option is object-fit: fill which we will appy to our img. This will make sure that the parent is always completely filled - stretching the image if necessary.

If you want to preserve your aspect ratio you can use object-fit: cover. The image will still fill the entire area, but does not distort, instead cropping the image at the borders

If you want to make soure the entire image is visible and don't care about filling the box, object-fit: contain will be most useful. The image will be fully visible and as large as allowed, but the side that is to small or large will be filled with empty space.

object-fit: scale-down is similar to object-fit: contain, but will never scale your image up, preventing ugly pixelated images.

Lastly there is also object-fit: none, which will keep your the original resolution. This will sometimes result in cropping your image and at other times leaving empty spaces on some or all sides. Of course this does not work if the width or height is set to be relative to that of its parent.
